# バスケットボールの教科書1 学習アプリ

**「バスケットボールの教科書1 技術を再定義する」** 著：鈴木良和  
インタラクティブな学習支援アプリケーション

## 📖 概要

本書の90ページをデジタル学習環境で効率的に習得するための学習アプリです。  
ブックマーク・メモ機能により、重要な箇所を記録し、学習効率を大幅に向上させます。

## ✨ 主な機能

### 📄 ページ閲覧
- 90ページの教科書画像をスムーズに表示
- 矢印キー（←→）またはボタンでページをスクロール
- 任意のページに直接ジャンプ可能
- レスポンシブデザイン（PC・タブレット・スマートフォン対応）

### 🔖 ブックマーク機能
- 重要なページをブックマーク
- ブックマーク時にメモを追加可能
- ブックマーク一覧から素早くページにジャンプ
- ブラウザのローカルストレージに自動保存

### 📝 メモ機能
- 各ページに詳細なメモを記録
- 内容と日付を自動保存
- メモの閲覧・編集・削除が容易
- メモ一覧で全体を把握

### ⌨️ キーボードショートカット
- **←** / **→** キー：ページ切り替え
- **ブックマークボタン** または右クリック：ブックマーク登録

## 🚀 使い方

### アプリへのアクセス
```
https://oogunishou-art.github.io/yosikazusuzuki2026/
```

### 基本的な操作

1. **ページを移動する**
   - 「← 前のページ」「次のページ →」ボタンをクリック
   - または矢印キーを使用

2. **特定のページに移動する**
   - ページ番号を入力ボックスに入力
   - 「移動」ボタンをクリック

3. **ブックマークを追加する**
   - 「⭐ ブックマーク」ボタンをクリック
   - （オプション）メモを入力

4. **メモを記録する**
   - 右側のサイドバーで「メモ」タブをクリック
   - テキストエリアにメモを入力
   - 「💾 メモを保存」ボタンをクリック

## 💾 データ保存について

- すべてのブックマークとメモは**ブラウザのローカルストレージ**に保存されます
- クラウドへの送信はなく、デバイス上のみに保存
- ブラウザのキャッシュをクリアするとデータが削除されます（注意！）

## 📋 必要な環境

- モダンブラウザ（Chrome、Firefox、Safari、Edge等）
- インターネット接続（初回読み込み時）
- 特別なインストール・セットアップは不要

## 📱 レスポンシブデザイン

- **PC**: 横並びレイアウト（ページビューア＋サイドバー）
- **タブレット・スマートフォン**: 縦積みレイアウト（フレキシブル対応）

## 🔧 技術仕様

- **フロントエンド**: 純粋なHTML5 + Vanilla JavaScript + CSS3
- **フレームワーク不要**: 単一HTMLファイルで完全に動作
- **ストレージ**: ブラウザのLocalStorage API
- **画像形式**: JPEG（90ページ、合計約16MB）

## 🆘 トラブルシューティング

### Q: ページ画像が表示されない
**A**: ブラウザのコンソール（F12キー）でエラーを確認してください。  
画像ファイルが `images/page-X.jpg` の形式で存在することを確認してください。

### Q: メモが保存されない
**A**: ブラウザのキャッシュ設定を確認してください。  
プライベートブラウジングモードではローカルストレージが無効になる場合があります。

### Q: 矢印キーが効かない
**A**: メモ入力中は矢印キーが使えません。  
ページビューアをクリックしてから使用してください。
